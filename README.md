# ResilienceAI: Supply Chain Risk Intelligence

ResilienceAI is a sophisticated risk management platform designed to provide real-time visibility and predictive simulation for global supply chain disruptions. Built for the modern enterprise, it combines multi-agent AI systems with dynamic data visualization to turn complex global signals into actionable intelligence.

## ğŸ Getting Started

### Prerequisites
- [pnpm](https://pnpm.io/)
- [Python 3.10+](https://www.python.org/)
- LLM API Keys (Gemini/Google)

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Chandi713/Hackathon---End-Of-The-World.git
    cd Hackathon---End-Of-The-World
    ```

2.  **Install dependencies**:
    ```bash
    pnpm install
    pip install -r ai_agents/requirements.txt
    ```

3.  **Run the application**:
    ```bash
    pnpm run dev:all
    ```
    *This command concurrently starts the Next.js dev server and the Python Agent server.*
